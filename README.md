## Awesome-online-ppt
## 在线实时演示文稿

----

局域网在线简单ppt演示，支持文字、预定义图片、淡入淡出、旋转等动画演示。


#### 一、准备
安装项目依赖包
```
yarn install
```
or
```
npm install
```



#### 二、开发者（Development）

配置地址
```
打开根目录config.js
将ip改成本机ip
```


启动项目

```
打开两个控制台

npm run api // 启动后端服务

npm start   // 启动前端
```

#### 三、部署发布（Production）
编译与打包
```
npm run build
```
